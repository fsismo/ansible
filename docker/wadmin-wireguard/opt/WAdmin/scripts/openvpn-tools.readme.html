Configuración cliente OpenVPN en Windows

Descarga / Instalación

Para poder conectarse a la vpn es necesario descargar el cliente según la versión de windows que utilice.

  • Para windows 64 bit.
  • Para windows 32 bit.

Una vez que finalizó la descarga realizar la instalación de la aplicación. El instalador va a solicitar el reinicio del equipo. Con el reinicio finaliza la instalación.

Configuración

El primer paso es ejecutar la aplicación. El nombre de la aplicación es "OpenVPN GUI".

La aplicación se aloja en el Área de notificación (Tray bar).

Sobre el ícono de OpenVPN, realizar clic derecho y seleccionar la opción "Import file ..." para importar el archivo servidor-usuario.ovpn provisto junto con este documento.

Una vez importado el archivo se puede proceder a realizar la conección con el servidor. Para ejecutar esta acción se debe realizar clic redecho sobre el ícono de la aplicación y seleccionar la opición "Connect".

Cuando el sistema solicite la contraseña con la que se encriptaron las llaves utiice "xpwdx" sin las comillas, si no desea que el sistema vuelva a pedir la contraseña tilde la opción "Save passowrd".
